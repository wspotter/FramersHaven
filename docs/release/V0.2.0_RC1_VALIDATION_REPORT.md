## Human Gates Before Final Windows Release

1. Run the generated ZIP on a clean Windows 10 or Windows 11 workstation.
2. Confirm first launch, dependency installation, browser opening, shutdown, restart, and database preservation.
3. Fill out `WINDOWS_TEST_REPORT_TEMPLATE.md` with machine details and screenshots.
4. Review generated accounting CSV files with the intended operator or bookkeeper.
5. Confirm license stance, pricing, support, refund, and distribution terms before accepting payment or making a broad commercial announcement.

## Rollback

The accounting export is edition-gated. If a release problem is found before final publication:

1. Keep Community as the default edition.
2. Stop distributing the release-candidate ZIP.
3. Revert the release-candidate commit or accounting feature merge on `main`.
4. Rebuild and re-run the full validation sequence before replacement.
