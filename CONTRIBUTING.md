# Contributing

1. Create a focused branch from `main`.
2. Use fictional fixtures and generated artwork only.
3. Add or update tests for behavior changes.
4. Run the complete verification commands in `README.md`.
5. Confirm `git status` contains no database, upload, export, backup, vendor, credential, or environment files.

Keep materials (`moulding`, `mat`, `glazing`) separate from shop services such as mounting, backing, printing, assembly, and royalties.
