Admin & Pricing

Admin owns catalog imports, manual material corrections, pricing rules, service prices, and local backups. Changes here affect future quote calculations.

Materials and services stay separate

Catalog materials

Moulding, mat, and glazing are catalog-backed items with SKU, cost, vendor, dimensions, preview image, and source metadata.

Shop services

Backing, mounting, frame mounting, various, assembly, royalties, and custom rows are admin-priced service options. Printing Options is a separate fixed-price list of print sizes: add a size, set its unit price, and turn it on when it is ready for quotes.

Do not import service work as catalog items.

The app rejects service-like catalog categories so material browsing and shop labor pricing cannot silently mix together.

Import vendor catalogs

  1. Prepare the source files. Put the current vendor CSV and preview ZIP files in the local pfd/ folder.
  2. Choose the matching import. Use PFD Mats or PFD Mouldings. A general CSV import is also available for the supported columns.
  3. Review the result. The import reports inserted, updated, skipped, duplicate, and preview counts.
  4. Spot-check the catalog. Search a few known SKUs and confirm names, costs, dimensions, and previews.

PFD imports upsert existing rows by SKU and category. Valid existing items are updated instead of duplicated; invalid rows are skipped.

Pricing and manual corrections

  • Pricing Rules controls the saved tax rate and material markups used by quote calculation.
  • Manual Pricing controls service cost, markup, pricing basis, label, and active state. Printing Options controls the selectable print sizes and fixed prices.
  • Catalog Editor can create a missing material or correct an imported item without rerunning the full import.
  • Use the cropped texture uploader to attach a better moulding strip preview to the selected moulding.
  • Calculate a fresh quote after changing pricing; an already calculated on-screen quote may need to be recalculated.

Backups and recovery

  1. Create a backup. Use Create Backup before major imports or workstation changes.
  2. Download the ZIP. The archive includes the database, uploads, exports, catalog snapshot, and backup metadata.
  3. Restore only while stopped. Stop the app before replacing studio.db or working directories from a backup.
  4. Verify after restore. Restart, check catalog counts, then spot-check recent jobs in Orders / Quotes.
Keep the current files before restoring.

Copy the existing database and working folders somewhere safe first. A restore should be reversible until the replacement is verified.
